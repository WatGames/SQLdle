My instagram is @wat_ethan and the link to my GitHub is here.

This project was started on the 10th of May 2026. When I started this I had no expirance with HTML, JS and CSS.

This project is a SQL based wordle clone, where you have to guess the correct SQL query in 6 tries or less. You can select from 3 difficulties, easy, medium and hard.
Each difficulty has an increasing level of difficulty relating to the complexity of the request and the difficulty of the queries.

Once you have selected a difficulty, you can press the "Play" button to start the game.

To go back to the main page, click the "Back" button below.
